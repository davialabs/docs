## Supported Framework (for now)

🔗 Currently, Davia only supports **Langgraph**.

Before using Davia, make sure your agent follows the following requirements.

## Requirements (current version)

✅ Your agent must be a **Langgraph**

✅ Your project structure **must include a langgraph.json file** to define dependencies and graphs.

✅ Davia must be installed in your environment, if it’s not the case just simply write the following command:

```bash
pip install davia
```

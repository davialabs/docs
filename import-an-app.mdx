## Test an app before importing it into your workspace

Clicking on an app card in the hub opens up a **playground** where you can test it.

<Card className="text-center" img="images/template-playground.png">
  Playground Window where you can test the app before making it your own
</Card>
The data that's being showed is mock data that was selected by the creator to showcase
the app's functionalities. To use this app you must import it into your workspace
using the **Use this app** button.

<Warning>
  When you import the app, the test data will be emptied so that you can start
  from fresh
</Warning>

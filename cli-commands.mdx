## Commands

| Command      | Purpose                                                                                                                  | Options                                                                                                                                                                  |
| ------------ | ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `davia init` | Initialize Davia in current directory. Creates `.davia` folder and project configuration.                                | • `--agent` - Specify AI coding agent (cursor, github-copilot, windsurf)                                                                                                 |
| `davia open` | Start web server and open browser. Finds available port if specified port is in use.                                     | • `--port` - Specify port number (default: 3005)                                                                                                                         |
| `davia push` | Upload documentation to remote workspace. Authenticates, links workspace, uploads files.                                 | None                                                                                                                                                                     |
| `davia docs` | Generate documentation using AI agent. Checks API key, runs agent, starts web server. Outputs files to `.davia/assets/`. | • `--overwrite` - Overwrite existing documentation files<br/>• `--no-browser` - Don't open browser after generation<br/>• `--port` - Specify port number (default: 3005) |
